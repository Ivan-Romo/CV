Portfolio

  • Chess Project

    A project to create a digital version of chess using Unity and coding in C#. The main objective was to faithfully recreate the experience of classic chess while exploring new implementation techniques that could enhance code efficiency and comprehension.


    One of the key technical decisions I made was to adopt a vector-based board representation instead of using a conventional matrix. This involved storing piece positions in a single vector, assigning each square on the board a unique index. This strategy significantly simplified game logic and facilitated rule implementation, while providing a solid foundation for future enhancements.


    The project encompassed standard features such as local multiplayer mode for two players and a single-player mode against a basic artificial intelligence. Although the AI did not reach an advanced level, it provided a playable experience for practicing and improving skills.

  • Mobile Game

    I set out to create a hyper-casual game for mobile devices. This project wasn't limited to programming and design; it was a comprehensive learning experience that included market analysis, mechanics adaptation, and marketing strategy.
